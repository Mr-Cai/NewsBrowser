# NewsClient-Kotlin
基于Kotlin+MVP+Retrofit+RxJava的新闻客户端
通过此项目你将学到
- 1.kotlin的大部分语法（包括类继承，kotlin中的Anko使用等）
- 2.kotlin类扩展
- 3.MVP设计模式
- 4.Retrofit+RxJava
- 5.Fragment懒加载架构
